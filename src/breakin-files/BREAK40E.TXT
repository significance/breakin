about Breakin: Breakin is another Breakout game ... 
               using 256 Colors , Soundblaster...
               with mouse-control only.....
	       Guess not bad ....	
               Play and see ...
	       
USING SOUNDBLASTER : - Make sure ct-voice.drv is in the path 
		     - start breakin with option /BLASTER AUTO
	               for default detection via ct-voice.drv
	               or /BLASTER PARAM <irq> <IO> or only /BLASTER.		      

HINT: If you run Breakin for the first time, enter SPEED-option after
      loading to adjust gamespeed.
      There is a Autodetectfunktion running. For manuel Change:
       1.) 'A' Turn off Autodetect
       2.) +/- adjust delay until STERO SET is on. 
     
      Leave with ESC or 'Q'.  The delay will be saved , so you only have
      to run it once.
